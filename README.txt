# Página web de Carta Interactiva

Este proyecto consiste en la creación de una página web interactiva con temática de carta dedicada a tu novio. La página principal contiene botones que llevan a otras páginas con dedicatorias especiales para transmitir tus sentimientos y emociones de una manera única y personalizada.

## Características principales

- Página principal con una introducción y opciones de dedicatorias.
- Tres páginas adicionales con dedicatorias especiales.
- Cada página incluye un mensaje personalizado y una imagen relacionada.

## Tecnologías utilizadas

El proyecto se ha desarrollado utilizando las siguientes tecnologías:

- HTML: Lenguaje de marcado utilizado para estructurar y presentar el contenido de las páginas web.
- CSS: Hojas de estilo utilizadas para definir el diseño y la apariencia de las páginas.
- JavaScript: Se ha utilizado JavaScript para la interactividad de los botones y redireccionamiento a las páginas de dedicatoria.

## Cómo usar el proyecto

1. Descarga el código fuente del proyecto o clona el repositorio.
2. Abre el archivo `index.html` en tu navegador web.
3. En la página principal, elige una de las opciones de dedicatorias haciendo clic en los botones correspondientes.
4. Serás redirigido a una página con una dedicatoria especial para tu novio.
5. Para volver a la página principal, haz clic en el botón "Volver" en cada página de dedicatoria.

Si deseas personalizar las dedicatorias o agregar tus propias imágenes, puedes modificar los archivos HTML correspondientes (`pagina1.html`, `pagina2.html`, `pagina3.html`) y reemplazar las imágenes existentes con las tuyas.
